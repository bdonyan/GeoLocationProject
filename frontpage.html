GeoLocation Project

by Brandon Yan, Incoming Junior for Mission San Jose 2021-2022 year

This is my GeoLocation Project! I built this for the Cisco 2021 High Shadow Program. My program takes in a JSON file input of an array of IP Address Objects. I made a GoLang code to randomly generate this file of IP Addresses. I use a NodeJS server to read in this file and display a HTML page of a Google Map. The points are converted to Longitude and Latitude with GoLang Code and an imported GeoLocation module. Then, a Javascript algorithm converts these into different color points on the map depending on density within a certain proximity of each individual point. Generate your own JSON file using the "Generate Input!" tab and plot the points with "Try It Yourself!" tab. A demonstration of the project is displayed on the right.
